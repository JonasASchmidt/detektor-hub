1. Import `Link` from `next/link` and `useState` from `react`
2. Replace the current `Collapsible` + `CollapsibleTrigger` pattern:
   - Use `useState` for each item's open state (or a single state object keyed by title)
   - Initialize open state based on whether the current pathname matches any sub-item URL (keep existing `defaultOpen` logic but convert to controlled state)
   - The `SidebarMenuButton` should be wrapped in a `Link` component pointing to `item.url` (which Plan 01 changes from "#" to real URLs)
   - Add an `onClick` handler on the `SidebarMenuButton` that toggles the collapsible open state: `onClick={() => setOpenState(prev => ({ ...prev, [item.title]: !prev[item.title] }))}`
   - Keep the `ChevronRight` icon with rotation animation for open state
   - Only show the collapsible chevron and sub-items if `item.items && item.items.length > 0`

3. Replace `<a href={subItem.url}>` with `<Link href={subItem.url}>` for sub-navigation items (prevents full page reloads)

4. Handle the active state: use `usePathname()` to highlight the current nav item. An item is "active" if `pathname.startsWith(item.url)` (for parent items) or `pathname === subItem.url` (for sub-items). Apply active styling via `isActive` prop on `SidebarMenuButton`.

**Structure per nav item:**
```tsx
<Collapsible open={openState[item.title]} onOpenChange={(open) => setOpenState(prev => ({ ...prev, [item.title]: open }))}>
  <SidebarMenuItem>
    <SidebarMenuButton asChild tooltip={item.title} isActive={pathname.startsWith(item.url)}>
      <Link href={item.url} onClick={() => setOpenState(prev => ({ ...prev, [item.title]: !prev[item.title] }))}>
        {item.icon && <item.icon />}
        <span>{item.title}</span>
        {item.items && item.items.length > 0 && (
          <ChevronRight className="ml-auto transition-transform duration-200" style={{ transform: openState[item.title] ? 'rotate(90deg)' : undefined }} />
        )}
      </Link>
    </SidebarMenuButton>
    {item.items && item.items.length > 0 && (
      <CollapsibleContent>
        <SidebarMenuSub>
          {item.items.map(subItem => (
            <SidebarMenuSubItem key={subItem.title}>
              <SidebarMenuSubButton asChild isActive={pathname === subItem.url}>
                <Link href={subItem.url}>
                  <span>{subItem.title}</span>
                </Link>
              </SidebarMenuSubButton>
            </SidebarMenuSubItem>
          ))}
        </SidebarMenuSub>
      </CollapsibleContent>
    )}
  </SidebarMenuItem>
</Collapsible>
```
